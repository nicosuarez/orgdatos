-------------------------------------------
DATA SAFE v1.0
-------------------------------------------
INSTALACION RAPIDA.

Bajar e instalar la libreria libreadline5-dev del gestor de paquetes.
Bajar e instalar la libreria imagemagick del gestor de paquetes.
Bajar e instalar la libreria libpng12-dev del gestor de paquetes.

1) Desde consola entrar al directorio DataSafe_Installer
2) Ejecutar ./install
3) Si las librerias estan bien instaladas, se tuvo que haber generado la carpeta DataSafe
4) Para correr el programa hay que ejecutar dentro del directorio DataSafe, ./DataSafe.

Atencion: si no se instala bien la libreria imagemagick si bien va a compilar cuando se agregue una
imagen jpg se puede tener resultados inesperados.
